# 使用 GUI

带你逛一圈这个应用 — 包括那些容易被错过的小功能。
每个设置都有提示: 把鼠标悬停在旁边的 ⓘ 图标上即可查看。

## 队列

- 使用**添加文件**按钮（旁边的文件夹图标可添加整个文件夹）添加视频或图像，
  或者直接把它们**拖放**到队列上。
- **拖动队列项的把手即可调整任务顺序** — 处理按从上到下的顺序进行。
- 每个队列项都有自己的按钮: **剪刀**按钮打开[区间编辑器](segments.md)，
  只修复该视频的一部分；叉号按钮把它从队列中移除。
- 处理过程中，每个队列项都会实时显示进度、FPS 和剩余时间。
- **清除已完成**移除已完成的任务；**清空**清空整个队列。

## 输出设置

- **与输入相同**会把每个结果写到原文件旁边。关闭它即可为所有输出
  选择同一个输出文件夹。
- **输出文件名模板**控制输出文件名 — `{original}` 代表输入文件名。
  如果某个模板会覆盖已有文件，受影响的队列项会立即高亮显示。
- **文件冲突**决定输出文件已存在时的处理方式: **自动重命名**
  （默认，安全）、**覆盖**或**跳过**。

## 预设

设置面板顶部的工具条可保存完整的设置组合。为你处理的每类内容创建
一个预设（例如“1080p 快速”和“4K 最佳质量”），一键切换。Jasna 会在
重启后记住你上次使用的预设。

## 每个视频的单独设置

在[区间编辑器](segments.md)中选择的检测模型和置信度会**按队列中的
每个视频单独记住**，因此同一队列中的不同视频可以使用不同的检测设置。

## 交互式图像修复

对于静态图像，**交互式处理图片**（位于 SD 1.5 设置部分）会打开一个
左右对比视图: 逐张浏览图片、尝试不同种子、对比原图、蒙版和修复结果，
只保存你喜欢的变体。比反复重跑整个任务来试验要快得多。

## 系统检查

每个版本首次启动时，Jasna 会运行系统检查（GPU、驱动、内存、安装
路径）。你随时可以从头部重新运行它 — 在更新驱动后或出现异常时很有用。

## 其他值得了解的功能

- **语言**: 在头部的下拉菜单中切换界面语言（重启后完全生效）。
- **日志**: 底部工具条的日志按钮显示实时日志，带级别过滤和导出按钮 —
  反馈问题时请附上导出的日志。
- **系统状态**: 处理时底部工具条会显示 GPU、VRAM、内存和 CPU 占用。
- **许可证**: 在头部输入你的支持者密钥；状态标签会显示是否已激活。
