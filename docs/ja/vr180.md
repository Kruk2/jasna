# VR180 動画

VR180 ファイルでは通常、左目用と右目用の映像が 1 枚の横長フレームに
並んでいます。Jasna はそのフレームを分割し、左右の目を別々に復元してから、
出力時に再び結合します。

## かんたん設定

1. 通常の動画と同じように VR 動画を追加します。
2. **VR180 モード**は**自動（推奨）**のままにします。
3. VR モザイクを最も正確に検出するには、**rfdetr-v5** または
   **zelefans-vr-yolo-v2** 検出モデルを使います。
4. 通常どおり処理を開始します。動画の一部だけを復元したい場合は、
   [区間エディター](segments.md)も使えます。

自動モードでは、高さが 1080 ピクセルを超える正確な 2:1 動画を必ず
サイドバイサイド VR として扱います — たとえば 3840x1920 や 8192x4096
です。互換性のある VR メタデータや既知の VR スタジオ名によって VR 処理が
有効になる場合もあります。

## モードの意味

- **自動（推奨）**: Jasna に判定を任せます。まずはこれを使ってください。
- **オフ**: 通常の平面動画として処理します。
- **SBS — 左右別処理**: 自動でファイルが認識されない場合に、
  サイドバイサイド VR 処理を強制します。
- **SBS + 魚眼**: VR 画像の端の近くでモザイクが大きく引き伸ばされている
  場合や、通常の SBS モードで見逃される場合に使います。このモードは
  レンズの歪みを一時的に補正し、検出と復元を改善します。

CLI での指定:

```bash
jasna --input input.mp4 --output output.mp4 --vr-mode auto
jasna --input input.mp4 --output output.mp4 --vr-mode sbs-fisheye
```
